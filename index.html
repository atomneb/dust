Welcome to the homepage for eblur/dust

This is a python library for calculating scattering absorption and scattering efficiencies for astrophysical dust from the infrared to the X-ray. This code can also be used to calculate dust scattering halo images in the X-ray, in both interstellar and intergalactic (cosmological) contexts.
